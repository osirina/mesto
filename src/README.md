## Описание проекта
Проект  Mesto - страница, где пользователь может делиться фотографиями своих путешествий.

## Что было сделано?
В рамках проектной работы были реализованы следующие решения:

* Flexbox
* Grid Layout
* CSS директивы
* Организация файловой структуры по БЭМ nested
* JS
